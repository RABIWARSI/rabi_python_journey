# BMI Calculator 🧮

A simple Python program to calculate Body Mass Index (BMI).

### 🔹 Features
- Takes weight (kg) and height (m) as input
- Calculates BMI
- Tells if you're Underweight, Normal, Overweight, or Obese

### 🧠 Skills Used
- Input handling
- If-else conditions
- Basic math in Python
